友链头像放在`/assets/link-img`，友链数据放在`/data/links.json`

link.json格式为：

```
[
    {
        "title": "小球飞鱼",
        "website": "https://mantyke.icu/",
        "image": "mantyke.png",
     "description": "我们会一起遇见鲸鱼吗？"
    },
	{
        "title": "友情链接2",
        "website": "",
        "image": "",
     "description": ""
    }
]
```
